# Lizenzen für externe Dependencies

Dieses Verzeichnis enthält alle Lizenzen für externe Bibliotheken und Ressourcen, die in diesem Projekt verwendet werden.

## Übersicht

- **common-template/**: Lizenzen für das common-template Modul
- **common-dto/**: Lizenzen für das common-dto Modul (wird in späteren Iterationen hinzugefügt)
- **common-entity/**: Lizenzen für das common-entity Modul (wird in späteren Iterationen hinzugefügt)
- **spark-web/**: Lizenzen für das spark-web Modul (wird in späteren Iterationen hinzugefügt)
- **spring-web/**: Lizenzen für das spring-web Modul (wird in späteren Iterationen hinzugefügt)

## Hinweise

Alle verwendeten Bibliotheken sind für kommerzielle Nutzung lizenziert und entsprechen den Anforderungen für wissenschaftliche Arbeiten.
